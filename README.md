# AxRaids
> Minecraft plugin to manage the default behaviour of minecraft raids
---
## Features
- Activate/Enable raids
---
## Installation
1. Download the latest release
2. Put the `.jar` into `/plugins` folder
3. Start the server
4. Stop the server
5. Edit the `config.yml`
6. Start the server
---
## Configuration
```
# config.yml
raids_enabled: false
```
